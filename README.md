The Enterprise Insights Hub project is a comprehensive Data Warehouse solution designed to integrate ERP and CRM datasets for efficient business analysis. The project leverages Medallion Architecture to structure data into Bronze (raw), Silver (cleaned), and Gold (aggregated) layers, ensuring data quality and performance.

The DWH is connected to Tableau to generate interactive dashboards with KPIs and actionable insights, enabling faster and informed decision-making for stakeholders.
